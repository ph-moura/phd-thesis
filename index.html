1. Description

This is a LaTeX class for the Ph.D. thesis.

2. Installation

To use locally just put the file phd-thesis.cls in the same directory as the document.

3. Usage

To load the class, use the following command: \documentclass[oneside]{phd-thesis}

By default, the a4paper and 12pt are passed. The [oneside] is optional. Some macros are defined

\documentclass[oneside]{phd-thesis}

% Title page properties 
\title[Título da tese]{Title of the thesis}	 
\author{Paulo H. de Moura}	 
\supervisor{Name of the Supervisor}	
\cosupervisor{Name of the Co-supervisor}  % Optional 
\institute[Nome do Instituto]{Institute Name}	 
\university[Nome da Universidade]{University Name}	 
\location{City}	
\logo{path/to/logo} % Default: align=left, width=2cm  

% Back cover text
\backcover{
	first text={
		Thesis presented to the \theinstitute~of the \theuniversity~in partial 
		fulfillment of the requirements for the degree of \uppercase{Master Jedi}, 
		in the area of \uppercase{Jedi Guardian}.
	},
	second text={
		Tese apresentada ao \theinstitute[2]~da \theuniversity[2]~como parte 
		dos requisitos exigidos para a obtenção do título de \uppercase{Mestre Jedi}, 
		na Área de \uppercase{Jedi Guardião}.
	}, 
	version text={
		Este trabalho corresponde à versão final da tese defendida pelo 
		aluno \theauthor~e orientada pelo Prof. Dr. \thesupervisor.
	}
}

% The document 
\begin{document}

\maketitle
\makebackcover

\dedication{To my son, Luke Skywalker.}
\epigraph{Master Yoda}{May the Force be with you!}

% Resumo
\begin{abstract}[name=Resumo]
 Escreva seu resumo aqui.
\end{abstract}

% Abstract
\begin{abstract}
 Write your abstract here.
\end{abstract}

% Sumário
\tableofcontents

\end{document}

These macros set the title, the author, the supervisor, the co-supervisor (optional), the name of the Institute, the name of the University, the city, the logo (optional) and the text for the cover page, respectively.

3.1 Guide for the commands

Title

The title and the subtitle, can be handled by the macros \title[]{} and \subtitle[]{}. Both macros have one mandatory and one optional argument. The optional argument is used to set the title (or subtitle) in a secondary language (by secondary language I mean the one that is not the used in the main part of the text). For example: if the text is written in english, one must set the portuguese as a secondary language. Once seted the title, its content can be obtained by means of the \thetitle command. To get the value passed through the optional argument of the title command, just use: \thetitle[2]. The same for the subtitle.
