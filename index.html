1. Description

This is a LaTeX class for the Ph.D. thesis.

2. Installation

To use locally just put the file phd-thesis.cls in the same directory as the document.

3. Usage

To load the class, use the following comand:

\documentclass[oneside]{phd-thesis}

By default, the a4paper and 12pt are passed. The [oneside] is optional. Some macros are defined

These macros set the title, the author, the supervisor, the co-supervisor (optional), the name of the Institute, the name of the University, the city, the logo (optional) and the text for the cover page, respectively.
